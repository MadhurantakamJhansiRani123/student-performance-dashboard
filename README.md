# 📊 Student Performance Dashboard (Power BI)

## 📌 Overview
This project is an interactive dashboard built using Power BI to analyze student performance.

## 🚀 Features
- KPI Cards (Total, Avg, Pass, Fail, Top Score)
- Subject-wise analysis
- Performance distribution
- Interactive slicers
- Dynamic titles & tooltips
- Reset filters button
